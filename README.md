# Russian Pronunciation Guide for Japanese Speakers

A static HTML/CSS site explaining Russian pronunciation to Japanese-speaking learners — with inline SVG sagittal diagrams showing tongue position for each sound.

**Live site:** [kiriosk.github.io/russian-pronunciation-for-japanese](https://kiriosk.github.io/russian-pronunciation-for-japanese/)

## Topics

| Page | Sounds | Focus |
|------|--------|-------|
| [Ж・З](zh-z.html) | [ʐ] [z] | Fricatives vs Japanese affricates |
| [Р・Л](r-l.html) | [r] [ɫ] | Trill and lateral vs Japanese flap |
| [Ф・В](f-v.html) | [f] [v] | Labiodental fricatives |
| [Ц・Щ](ts-shch.html) | [ts] [ɕː] | Affricate and alveopalatal fricative |
| [Ы・母音](vowels.html) | [ɨ] | Back unrounded vowel and vowel reduction |
| [軟子音](soft.html) | Cʲ | Palatalization (hard vs soft consonants) |

## Tech

- Pure HTML + CSS, no JavaScript, no build step
- Inline SVG anatomical diagrams (sagittal cross-sections)
- Hosted on GitHub Pages
