格

格の世界

日本語の助詞で理解するロシア語の格変化

1か月目

もう知っている仕組み

まず、普通の日本語の文を見てみましょう。

私は　本を　読む

「は」があるから、「私」が話題だとわかる。「を」があるから、「本」が読む対象だとわかる。

つまり、助詞が単語の役割を教えてくれるわけです。助詞がなかったら、誰が何を読むのかわかりません。

実は、ロシア語もまったく同じことをしています。ただし、やり方が少し違います。

ロシア語も同じことをしている

Я чита́ю кни́гу.
私は本を読む。

日本語では、単語の後ろに助詞を付けます。ロシア語では、単語の語尾を変えます。

本を
助詞を後ろに付ける
→
кни́гу
語尾を変える（книга → книгу）

「книга」（本）の語尾「-а」が「-у」に変わりました。この「-у」が日本語の「を」と同じ働きをしています。

このように語尾を変えて単語の役割を示す仕組みを「格（かく）」と呼びます。

6つの格 ＝ 6つの助詞グループ

ロシア語には格が6つあります。それぞれが日本語の助詞に対応しています。

1. は／が → 主格（Именительный）

学生が
学生が読む
→
студе́нт
語尾変化なし（辞書の形のまま）
Студе́нт чита́ет.

2. を → 対格（Винительный）

本を
本を読む
→
кни́гу
-а → -у
Чита́ть кни́гу.

3. の → 生格（Родительный）

学生の
学生の本
→
студе́нта
— → -а
Кни́га студе́нта.

4. に → 与格（Дательный）

友達に
友達にあげる
→
дру́гу
— → -у
Дать дру́гу.

5. で／と → 造格（Творительный）

ペンで
ペンで書く
→
ру́чкой
-а → -ой
Писа́ть ру́чкой.

6. で（場所）／に → 前置格（Предложный）

学校で
学校で勉強する
→
в шко́ле
-а → в + -е
Учи́ться в шко́ле.

完璧に一致するわけではない

注意

上の対応はあくまで「だいたい同じ」です。日本語の「に」がロシア語では与格になることもあれば、前置格になることもあります。逆に、一つの格がいくつかの日本語の助詞に対応する場合もあります。

でも心配しないでください。ここでは「ロシア語の格 ＝ 日本語の助詞と同じ考え方」という感覚がつかめれば十分です。

次のレッスン7〜11で、それぞれの格をひとつずつ詳しく学んでいきます。

練習：助詞と格をマッチしよう

練習問題

次の日本語の文で太字の助詞は、ロシア語のどの格に対応するでしょうか？

  1. 猫が寝ている。
  2. 水を飲む。
  3. 母の料理がおいしい。
  4. 子供にプレゼントをあげる。
  5. 箸で食べる。
答えを見る
  1. 「が」→ 主格（Именительный）— 主語を示す
  2. 「を」→ 対格（Винительный）— 目的語を示す
  3. 「の」→ 生格（Родительный）— 所有を示す
  4. 「に」→ 与格（Дательный）— 相手を示す
  5. 「で」→ 造格（Творительный）— 道具を示す
形容詞の基本
対格＝「を」
